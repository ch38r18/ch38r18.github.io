添加相册

hugo添加相册页面

功能：把指定文件夹的图片展示在页面上

日常操作：把图片放到 static/photos 中即可，搭配 Github Action 自动化部署

static
└── photos
    ├── 1.jpeg
    ├── 2.jpeg
    └── 3.jpeg

集成到我的主题中

  • 把照片放入 static/photos
  • 创建content/photos/_index.md
---
title: "⚛ 我的相册"
layout: "photos"
---
  • 新建 layouts/_default/photos.html
{{ define "main" }}
<div class="post">
  <h2 class="post-title">{{.Title}}</h2>
</div>

<div class="page-photos">
  {{ range (readDir "./static/photos") }}
  <figure>
    <img src="{{"photos/" | absURL }}{{ .Name }}" alt="{{ .Name }}" /> alt="{{ .Name }}" />
    <figcaption>{{ .Name | replaceRE "(.*)[.].*" "$1"}}</figcaption>
  </figure>
  {{ end }}
</div>
{{ end }}

  • 方法一：css设置图片样式，新建 static\css\photos.css

    .page-photos figure img{
  max-width:80%;
  margin:0 auto 3rem;
}

  • 方法一：修改layouts/_default/photos.html

    ...
    <figcaption>{{ .Name | replaceRE "(.*)[.].*" "$1"}}</figcaption>
    <link rel="stylesheet" href="{{ "css/photos.css" | absURL }}">
  </figure>
...

  • 方法二：在layouts/_default/photos.html直接设置图片的样式代码

...
</div>

<style>
img {
    max-width: 100%;
    height: auto;
}
</style>

<div class="page-photos">
...
